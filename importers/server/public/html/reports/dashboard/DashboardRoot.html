Welcome to the "See Your Business Dashboard"

The purpose of this site is to provide dynamically generated graphics that show trends based on the concepts presented on the for Entrepreneurs.com web site.

The eventual support will provide two methods of data input. Live through this browser interface, and via a web service request that will return a url to a generated set of reports that can be further refined dynamically.

A overview of the concept can be seen in a sequence of pages starting here.

Instead of trying to anticipate what you want in a dashboard or how you want to organize it, a set of individual pages/reports are produced and you can leverage tehm in your own portal or dashboard. In the following links you will see some example overview pages that leverage the report page data and then link to the generated pages with more detailed views and data visible.
